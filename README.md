# my-first-repo

## Setup

Create and activate a virtual environment:

```sh
conda create -n my-first-env-2025 python=3.11

conda activate my-first-env-2025
```

Install packages:

```sh
# pip install pytest

pip install -r requirements.txt
```

## Usage

Play a game of rock, paper scissors:

```sh
python app/rps.py

python -m app.rps
```

## Tests

Run the tests:

```sh
pytest
```
